4 types of Positions in CSS
------------------------------------------------------------------------------------

Static

Relative

Absolute

Fixed

------------------------------------------------------------------------------------
------------------------------------------------------------------------------------
Static
------------------------------------------------------------------------------------
HTML default flow;


h1 {

position : static;
left: 50px;
top: 50px;

}




------------------------------------------------------------------------------------
------------------------------------------------------------------------------------
Relative
------------------------------------------------------------------------------------
Position relative to default position
Move a Div from its relative position to a new Position starting from its current position.

.class1 {

    position: relative;
    left : 50 px;
    top : 50 px;
}







------------------------------------------------------------------------------------
------------------------------------------------------------------------------------
Absolute
------------------------------------------------------------------------------------




.class1 {
    
    position: absolute;
    left : 50 px;
    top : 50 px;
}




------------------------------------------------------------------------------------
------------------------------------------------------------------------------------
Fixed
------------------------------------------------------------------------------------



.class1 {
    
    position: fixed;
    left : 50 px;
    top : 50 px;
}
